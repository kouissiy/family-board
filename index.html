⚙️ スプレッドシート設定

📋 スプレッドシートの書き方
シート1（データ）に以下の形式で入力：

A列：項目名　B列：内容

message → 家族からのメッセージ本文
message_from → 差出人（例：こういちより）
photo_url → 写真のURL（Google ドライブ共有リンク等）
schedule_月 → 月曜の予定
schedule_火 → 火曜の予定（ヘルパーさん来訪など）
schedule_水〜schedule_日 → 各曜日の予定
shop_食品_牛乳 → 食品カテゴリの「牛乳」（shop_カテゴリ_品名）
todo_薬の確認をする → ヘルパーさんへのTODO
health → 体調・健康メモ
handover → 前回からの申し送り事項
